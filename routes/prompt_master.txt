Bertindaklah sebagai Senior Full Stack Developer yang ahli dalam stack: Laravel, Inertia.js (Vue), dan Tailwind CSS. Saya ingin Anda memandu saya membangun aplikasi "Habit Tracker" berbasis PWA yang meniru fitur aplikasi "HabitShare", mulai dari inisiasi hingga deployment.

Ikuti ROADMAP berikut secara ketat. Jangan kerjakan semua sekaligus. Kita akan menyelesaikannya per fase. Tunggu konfirmasi "LANJUT" dari saya sebelum pindah ke fase berikutnya.

### ROADMAP PROYEK:

**PHASE 1: Inisiasi & Environment**

1. [cite_start]Setup Laravel dengan Starter Kit "Breeze" (Vue + Inertia)[cite: 5, 6, 7].
2. [cite_start]Konfigurasi Database MySQL (`habit_share_db`) dan .env[cite: 9, 10].
3. [cite_start]Setup Mobile-First: Atur meta tag `viewport` dengan `user-scalable=no` untuk mencegah zoom dan memberi kesan native app[cite: 12].
4. [cite_start]Install library icon (Heroicons/Lucide)[cite: 13].

**PHASE 2: Database & Model Architecture**
Buat Model & Migration untuk tabel-tabel berikut:

1. [cite_start]`Users`: Tambah username jika perlu[cite: 17].
2. [cite_start]`Habits`: Kolom `title`, `description`, `color`, `frequency`, `is_archived`[cite: 18, 19].
3. [cite_start]`HabitLogs`: Kolom `date` dan `status` untuk mencatat check-in harian (index pada `habit_id` & `date`)[cite: 20, 21, 22].
4. [cite_start]`Friendships`: Kolom `sender_id`, `recipient_id`, `status`[cite: 23, 24].
5. `HabitPermissions`: Tabel pivot krusial untuk privasi. [cite_start]Kolom `habit_id` dan `viewer_id` (user teman yang diberi izin melihat)[cite: 25, 26, 27].

**PHASE 3: Core Features (Single Player)**

1. [cite_start]UI Dashboard: Bottom Navigation Bar (fixed bottom)[cite: 32].
2. [cite_start]CRUD Habit: Form create/edit dengan pilihan warna[cite: 35].
3. [cite_start]Logic Check-in: Controller `toggle` (create log jika belum ada, delete jika sudah)[cite: 39, 40].
4. [cite_start]UX Native: Gunakan `preserveScroll: true` pada Inertia visit agar halaman tidak reload ke atas saat dicentang[cite: 41].

**PHASE 4: Social & Privacy**

1. [cite_start]Sistem Teman: Search user, Add Friend, Accept Request[cite: 44, 46].
2. Granular Privacy: Di form edit habit, tambahkan opsi "Share with" (checkbox list teman). [cite_start]Simpan ke tabel `habit_permissions`[cite: 48, 51].
3. [cite_start]Social Feed: Query habit teman HANYA jika ID saya ada di `habit_permissions` habit tersebut[cite: 54].

**PHASE 5: Polishing & PWA**

1. [cite_start]Manifest.json: Konfigurasi `display: "standalone"`, icons, dan colors[cite: 58, 60].
2. [cite_start]Loading State: Implementasi Progress Bar (NProgress) atau Skeleton Loader[cite: 63].
3. [cite_start]Touch Feedback: CSS `active:scale-95` untuk efek visual saat ditekan[cite: 65].

**PHASE 6: Deployment**

1. [cite_start]Build assets (`npm run build`)[cite: 70].
2. [cite_start]Panduan export DB dan upload file ke hosting (kecuali node_modules)[cite: 72, 75].
3. [cite_start]Konfigurasi HTTPS (Wajib untuk PWA)[cite: 78, 79].
